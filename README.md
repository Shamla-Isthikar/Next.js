# Next.js

A comprehensive Next.js 16 project demonstrating modern web development practices using:

- App Router
- Server Components
- Client Components
- Nested Layouts
- Dynamic Routing
- TypeScript
- Tailwind CSS
- Full-Stack Development Concepts

## Features

- File-based Routing
- Reusable Components
- Server-side Rendering (SSR)
- Client-side Interactivity
- Responsive UI Design
- Modern Next.js Architecture

## Technologies

- Next.js 16
- React
- TypeScript
- Tailwind CSS

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Shamla-Isthikar/Next.js.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
